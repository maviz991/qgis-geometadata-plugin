    # Versão 1.0.0 (Data de Lançamento) - Release de Produção
    ## Melhorias de Usabilidade e Qualidade de Vida (UX)
    - **Login Persistente:** O estado de login agora persiste durante toda a sessão do QGIS, eliminando a necessidade de autenticações repetidas.
    - **Proteção Contra Perda de Dados:** Um aviso de confirmação é exibido ao tentar fechar a janela com alterações não salvas.
    - **Busca de Camadas Aprimorada:** A janela "Associar Camada" agora possui um campo de busca com autocompletar, facilitando a localização de camadas no GeoServer.
    - **Botão "Hoje":** Adicionado um botão de atalho para preencher a data do metadado com a data e hora atuais.
    - **Link para o Manual:** Adicionado um link de acesso rápido ao manual do usuário na interface principal.
    ## Robustez e Qualidade dos Dados
    - **Validação de Campos Obrigatórios:** O plugin agora impede a exportação de metadados incompletos, informando ao usuário quais campos essenciais precisam ser preenchidos.
    - **Limpeza Inteligente de Dados:** Os campos de "Palavras-chave" e "Escala" agora normalizam a entrada do usuário, aceitando diferentes formatos (ex: ";" como separador ou "1:5.000" para escala).
    - **BBox Dinâmico:** A Extensão Geográfica (Bounding Box) é sempre atualizada com base na camada ativa no QGIS, garantindo que o metadado reflita o estado real dos dados.
    - **Ciclo de Vida do UUID:** Melhorada a lógica de criação e atualização de metadados no Geohab para garantir a consistência do UUID do registro.
    - **Prevenção de Contatos Duplicados:** O gerador de XML agora evita a duplicação de informações de contato quando o preset "CDHU" é selecionado.
    ## Manutenção e Configuração
    - **Contatos Externalizados:** Os presets de contato foram movidos para um arquivo `contacts.json`, permitindo fácil atualização sem a necessidade de editar o código do plugin.
    ## Interface (UI)
    - **Tema Adaptativo:** A interface agora detecta automaticamente o tema do QGIS (claro ou escuro) e ajusta suas cores para uma integração visual perfeita.
    - **Melhorias de Layout:** Ajustes gerais na interface para melhor responsividade e consistência visual.
    # Versão 0.0.9 (Data) - Pré-release

    - Exportação para arquivo XML
