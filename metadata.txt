    <b>FUNÇÕES:</b>
    1. Criação e edição de metadados MGB 2.0;
    2. Preenchimento automático de título e BBOX;
    3. Persistência de dados (sidecar/banco de dados); 
    4. Exportação para XML;
    5. Continuar depois - Salve metadado associado a uma camada;
    6. Asssociação de GeoServiços WMS e WFS;
    5. Publicação direta no Geohab.

    <b>UX:</b> 
    • Login persistente na sessão QGIS; 
    • Proteção contra perda de dados ao fechar; 
    • Busca de camadas aprimorada com autocompletar; 
    • Botão "Hoje" para data do metadado.

    <b>ROBUSTEZ:</b> 
    • Validação de campos obrigatórios; 
    • Limpeza inteligente de dados (palavras-chave, escala);
    • BBox dinâmico sempre atualizado pela camada;
    • Consistência de UUID na criação/atualização;
    • Prevenção de contatos duplicados no XML. 
        
    <b>INTERFACE:</b> 
    • Tema adaptativo (claro/escuro) automático;
    • Link para o manual do usuário.
